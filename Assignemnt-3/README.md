# 🎓 Student Scoreboard Dashboard



A sleek, state-driven academic management dashboard built with **React**. This project demonstrates real-time data processing, dynamic UI updates, and modern CSS grid styling with a "Cyber-Grid" aesthetic.

## 🌟 Key Features

- **Real-time Analytics:** Automatically calculates Total Students, Pass/Fail counts, and Class Average instantly.
- **Dynamic CRUD Operations:**
    - **Create:** Add new students with auto-validation.
    - **Update:** Edit scores in-line with immediate grade recalculation.
- **Automated Grading System:** - Logic-based grading (A+, B, D, F).
    - Status toggles (Pass/Fail) based on performance thresholds.
- **Modern UI/UX:** High-contrast dark mode featuring a custom CSS grid background and neon accents.

## 🛠️ Tech Stack

- **Frontend:** React.js (Functional Components)
- **State Management:** React Hooks (`useState`, `useEffect`)
- **Styling:** CSS3 (Advanced Grid Layouts & Flexbox)
